Instruções para usar o script de gerador de senha.

	1. Baixar e installar NodeJS em seu computador. Link: https://nodejs.org/en/

	2. Abra o cmd (Prompt de Comando) e acesse o caminho da pasta do index.js gerador de senha.

		Ex: cd Desktop/nome_da_pasta/

	3. Execute o script.js pelo seguinte comando:

		Ex: node index.js nome_usuario qtd_chars_password

	4. O usuario e senha sera enviado para um arquivo texto accounts.txt que se encontra na mesma pasta do index.js

		Obs: Se o arquivo nao existir, sera criado e para eventuais usous, suas contas serão salvas no arquivo!
